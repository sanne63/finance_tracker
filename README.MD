# Personal Finance Tracker 💰
A command-line application to track personal income and expenses, built with Python and SQLite.

## Features
- Add income and expense transactions
- View all transactions in a formatted table
- Update existing transactions
- Delete transactions
- View balance summary (total income, expenses, and balance)

## Tech Stack
- **Language:** Python 3
- **Database:** SQLite (built-in)
- **Interface:** Command Line Interface (CLI)

## Project Structure
finance-tracker/
├── database.py   # Database connection and initialization
├── models.py     # CRUD operations
├── Main.py       # Main application and CLI menu
└── README.md     # Project documentation

## How to Run
1. Make sure Python 3 is installed on your machine
2. Clone this repository
3. Navigate to the project folder
4. Run the application

## Usage
After running the app, you will see a menu:
========================================
PERSONAL FINANCE TRACKER

Tambah transaksi
Lihat semua transaksi
Update transaksi
Hapus transaksi
Ringkasan saldo
Keluar
========================================
Select an option by entering the corresponding number.
